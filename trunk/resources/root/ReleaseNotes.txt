Welcome to Pondskum 1.0.3!
==========================

New features and Fixes
======================

1. Removed Guice and Scala dependencies to make the distribution file smaller.
2. Added new usage formatting to trayicon as well as tablet guis.
3. Status information is now transferred between guis on the mvc.
4. Added context menues to switch between the progression bar, tablet and trayicon guis in the mvc.
5. Added a dialog to display errors that occur while installing the trayicon.
6. Added a splash screen.
7. The tablet now automatically scrolls to the latest row.
8. Removed superfluous exist button from tablet.
9. Tablet data can now be sorted.
10. Moved totals of the tablet gui into it's own more visible panel.
